# República Justa

Videojuego arcade de temática social y política en el que un personaje bueno se enfrenta a la corrupción a lo largo de 10 niveles.

## Objetivo del proyecto

El objetivo de este proyecto es practicar desarrollo web y lógica de juego mediante una experiencia arcade sencilla, directa y con un mensaje de fondo: representar, desde el humor y la acción, la lucha contra la corrupción.

Forma parte de una serie de pequeños proyectos creados como práctica de programación, diseño de interacción y experimentación creativa.

## Características principales

- Videojuego arcade con 10 niveles.
- Temática política y social tratada desde un enfoque lúdico.
- Progresión por niveles.
- Mecánica centrada en superar obstáculos y enfrentarse a la corrupción.
- Proyecto ligero orientado a navegador.

## Tecnologías utilizadas

- HTML
- CSS
- JavaScript

## Estado del proyecto

Proyecto funcional en fase inicial de aprendizaje y mejora continua.

## Aprendizajes trabajados

Con este proyecto se practican conceptos como:

- Estructura básica de un juego web.
- Lógica de niveles.
- Control de eventos e interacción con el usuario.
- Organización de una experiencia arcade sencilla.
- Uso de JavaScript para crear dinámicas de juego.
